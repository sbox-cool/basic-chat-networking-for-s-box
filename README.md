# Network Storage Example: Chatbox with database storage
This s&box project contains working collection, endpoint and workflow examples that you can publish to your own Network Storage project.

### Prerequisites
- https://sbox.game/sboxcool/network-storage (recommended)
or
- https://github.com/sbox-cool/sbox-network-storage (manual install + documentation)
- 
### Features 
- Send message
- Periodically get messages from server
- Player storage

NOT production ready. This is just an example to show how Network Storage system works on a basic level to get people started. This project for instance would benefit from a rate limiter configured in the sbox.cool/network-storage Rate Limit tool
- Not tested yet for multiplayer etc.
